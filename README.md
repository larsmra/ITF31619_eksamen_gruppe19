# ITF31619_eksamen_gruppe19

For å kjøre prosjektet:

- npm install (både i frontend og backend)
- npm run dev (både i frontend og backend)
- starte mongo.exe og skriv "use eksamen" 

For å opprette en admin bruker:

	Dette kan gjøres gjennom postman:
	{
		"role":
		"name":
		"email": 
		"password":
		"adminPassword":
	}

-Bruk verdiene til EMAIL_ADMIN, ADMIN_PASSWORD og som er til stedet i server/.env for email og adminPassword

Oppgave 6:
Endre verdiene i .env for EMAIL_HOST, EMAIL_PORT, EMAIL_USER, EMAIL_PASSWORD tilpasset din mailtrap bruker

Store deler av server koden er inspirert fra forelesnings notater. 
Der tilnærmet ren kode et hentet er det lagt med en kommentart i filen som henviser til gitt forelesning.
